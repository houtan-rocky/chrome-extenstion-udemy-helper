# Chrome Web Store Assets - Screenshot Guide

## How to Screenshot These HTML Files

1. **Open the HTML file in Chrome**
   - Double-click the HTML file, or
   - Right-click → Open with → Google Chrome

2. **Open Chrome DevTools**
   - Press `F12` or `Ctrl+Shift+I` (Windows/Linux) / `Cmd+Option+I` (Mac)
   - Or right-click → Inspect

3. **Set Device Dimensions**
   - Click the device toolbar icon (or press `Ctrl+Shift+M`)
   - Enter the exact dimensions:
     - **Store Icon**: 128 x 128
     - **Small Promo Tile**: 440 x 280
     - **Marquee Promo Tile**: 1400 x 560
     - **Screenshot 1**: 1280 x 800
     - **Screenshot 2**: 1280 x 800
     - **Screenshot 3**: 640 x 400

4. **Take Screenshot**
   - Press `Ctrl+Shift+P` (Windows/Linux) / `Cmd+Shift+P` (Mac)
   - Type "screenshot" and select:
     - **"Capture screenshot"** - for full page
     - **"Capture node screenshot"** - for selected element
   - Or use the three-dot menu → More tools → Screenshot

5. **Save the Screenshot**
   - The screenshot will be saved to your Downloads folder
   - Rename it appropriately (e.g., `small-promo-tile.png`)

## Files Included

- `store-icon.html` - 128x128 store icon
- `small-promo-tile.html` - 440x280 promo tile
- `marquee-promo-tile.html` - 1400x560 marquee tile
- `screenshot-1.html` - 1280x800 main course page view
- `screenshot-2.html` - 1280x800 extension popup view
- `screenshot-3.html` - 640x400 mobile/compact view

## Tips

- Make sure the browser zoom is at 100% (not zoomed in/out)
- Use the exact dimensions specified above
- Screenshots should be saved as PNG format
- Ensure the page is fully loaded before taking the screenshot
