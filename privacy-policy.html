Privacy Policy for Udemy Course Insights

Last Updated: December 2024

Introduction

Udemy Course Insights ("we", "our", or "the extension") is committed to protecting your privacy. This Privacy Policy explains how our Chrome extension handles information when you use it.

Data Collection and Usage

What We Do NOT Collect

We do NOT collect, store, or transmit:

  • Personal information (name, email, address, etc.)
  • Authentication credentials or passwords
  • Browsing history
  • User activity data (clicks, keystrokes, etc.)
  • Location information
  • Financial or payment information
  • Health information
  • Personal communications

What We Access

The extension accesses the following information to function:

  1. Current Tab URL: The extension reads the URL of the currently active browser tab to:
    • Detect when you're viewing a Udemy course page
    • Extract the course ID from the URL
    • Display relevant course information
  2. Page Content: The extension reads page content on Udemy course pages to:
    • Find where to display the creation date
    • Identify course title elements
  3. Course Metadata: The extension fetches publicly available course metadata (specifically creation dates) from Udemy's public API. This data is:
    • Publicly accessible information
    • Not associated with your personal account
    • Only fetched when you visit a course page

How We Use This Information

  • URL Reading: Used only to identify Udemy course pages and extract course IDs
  • Page Content: Used only to display the creation date in the appropriate location
  • API Calls: Made directly from your browser to Udemy's servers to fetch course creation dates

Data Storage

  • No Local Storage: The extension does not store any data locally on your device
  • No Remote Storage: The extension does not send data to any external servers
  • No Cookies: The extension does not use cookies or tracking mechanisms

Data Sharing

We do NOT share, sell, or transmit any data to:

  • Third-party services
  • Analytics platforms
  • Advertising networks
  • Any external servers

All data processing happens entirely within your browser.

Permissions Explained

Host Permissions (https://*.udemy.com/*)

This permission allows the extension to:

  • Access Udemy course pages
  • Fetch course metadata from Udemy's public API
  • Display creation dates on course pages

The extension only accesses Udemy domains and does not access any other websites.

Tabs Permission

This permission allows the extension to:

  • Read the URL of the currently active tab
  • Detect when you're viewing a Udemy course page

The extension does NOT:

  • Modify tabs
  • Create new tabs
  • Access tab content beyond reading the URL
  • Track your browsing history

Third-Party Services

The extension makes API calls to:

  • Udemy's Public API (https://www.udemy.com/api-2.0/courses): To fetch publicly available course creation dates

These API calls are made directly from your browser to Udemy's servers. We do not act as an intermediary or store any data from these calls.

Your Rights

You have the right to:

  • Disable the extension at any time through Chrome's extension management
  • Uninstall the extension at any time
  • Review this privacy policy to understand how your data is handled

Children's Privacy

This extension is not intended for children under 13. We do not knowingly collect any information from children.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date.

Contact

If you have questions about this Privacy Policy, please contact us through the extension's support channels.

Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • Chrome Web Store User Data Privacy policy
  • General data protection best practices

Summary

This extension reads URLs and page content to display course creation dates. No personal data is collected, stored, or transmitted. All processing happens locally in your browser.
